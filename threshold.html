Thresholding 🎚️

The simplest way to count — a single brightness cutoff.

How it works

Every pixel brighter than the threshold is counted as part of an object; everything darker is background. Touching objects that share bright pixels are counted as one — for crowded or overlapping cells, try the Cellpose or Classifier tools instead.

Instructions

1
Load images
Drag microscopy images (.tif, .png, .jpg) onto the canvas. Each is auto-thresholded on load.
2
Adjust
Leave it on Auto (Otsu picks the cutoff per image), or drag the slider for a manual cutoff applied to every image.
3
Read the count
The object count updates live. Tick Invert if your objects are darker than the background (e.g. brightfield).
