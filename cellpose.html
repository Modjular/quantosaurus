Cellpose 🔬

A pretrained deep-learning model for tough, crowded cells — running in your browser.

When to use it

Cellpose is the vanilla cyto3 model — the same weights the desktop tool ships with, no training required. Reach for it when cells touch or overlap and thresholding merges them into blobs. It's the heaviest option: the model is ~26 MB (downloaded once, then cached) and each image takes a few seconds to run.

Instructions

1
Load images
Drag your microscopy images (.tif, .png, .jpg) onto the canvas.
2
Set the cell size
Enter the approximate cell diameter in pixels. This is the single most important setting — get it roughly right and the rest usually takes care of itself.
3
Press Run
Every cell is outlined and counted. Touching cells stay separate — that's what Cellpose is for.

Which channel do I pick?

If your image is a single channel, you're already done — leave it as-is and press Run. Picking a channel never breaks anything; at worst it changes the result, and you can switch and re-run freely. For two-channel images, set Cytoplasm to the channel showing the cell body and Nucleus to the nuclear stain (or leave Nucleus as "None"). Cellpose was trained on cytoplasm+nucleus pairs, but works well on a single channel too.
